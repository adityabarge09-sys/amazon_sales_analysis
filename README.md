📊 Amazon Sales Dashboard (Power BI)

📌 Project Overview

This project is a Power BI dashboard built using Amazon sales data. It provides meaningful insights into sales performance, product categories, revenue trends, and customer behavior.

The main objective is to analyze sales data and present it visually to support better business decisions.

🛠️ Tools & Technologies Used

Power BI (.pbix)

Data Cleaning & Transformation

Data Visualization

DAX (Data Analysis Expressions)

📂 Dataset Description

The dataset includes the following:

Product details

Sales amount

Order date

Customer information

Category & sub-category

📅 Sales trends over time

🛍️ Top-selling products and categories

💰 Revenue analysis
